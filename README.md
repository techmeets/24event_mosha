# 【模写課題イベント】ソースコードの共有

## 説明

2020/10/24 日に開催された模写課題イベントでのソースコードになります。

## ファイル内構成

CSS ファイルはなく、HTML ファイル内の style タグで記述されています。
コピペでも可能ですが、余裕がある方は、CSS と HTML ファイルを分けて記述することをおすすめします。
